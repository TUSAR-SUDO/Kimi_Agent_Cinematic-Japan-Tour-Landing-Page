# Japan Tours — Cinematic Landing Page

A cinematic scroll-driven landing page for a premium 10-day Japan tour experience across Osaka, Kyoto, and Tokyo.

## Live Demo

https://kimi-agent-cinematic-japan-tour-lan.vercel.app/

## Preview

A premium editorial-style landing page featuring:

- Layered parallax hero section
- Smooth Lenis scrolling
- Framer Motion animations
- Glassmorphism UI

- Accessibility support

## Installation

Clone the repository:

```bash
git clone https://github.com/TUSAR-SUDO/Kimi_Agent_Cinematic-Japan-Tour-Landing-Page.git
```

Go into the project:

```bash
cd Kimi_Agent_Cinematic-Japan-Tour-Landing-Page/app
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
